Basketball Timekeeper is an iOS application that keeps track of an ongoing basketball game.


Includes the ability to create rosters and players as well as see a log of all games played.


Pre-game set up allows you to pick created teams that are in a Swift Realm Database, set halves or quarters, and half/quarter length.


In-game it allows you to set team names, add points, call timeouts and fouls, and start/stop the game clock.



Pass the Class is a 2D platformer videogame created using Unity and C# where players traverse school themed levels such as math and chemistry class.


Throughout the levels there are enemies (books, equations, beakers) that the player must avoid using double jumps and running. To get to the next level players must reach the bell and ring it signifying the end of class.


Along the way there are collectables such as coins and treausure chests that increase your score so you can compete against other players.
