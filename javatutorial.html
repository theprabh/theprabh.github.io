Java Tutorial

Syntax

Syntax is basically the way the language looks and works.

For instance we have the following rules when it comes to syntax:

  • Case Sensitivity - Java is case sensitive which means that upper and lower case letter have different meaning. For example Car and car would have a different meaning.
  • Method Names - Method names should start with a lower case letter.
  • Class Names - Class names should start with an upper case letter.
  • Main Method - public static void main(String args[]) is known as the main method. It is a required part of every java program.
  • Semicolons - Every statement ends with a semicolon ;
  • Comments - Comments in java are done by using // followed by whatever you want to write. These sentences aren't used in the actual program, but they can be used to help write a program as it makes things easier to understand
  • File Name - The name of the file should be the same as the class name.

Data Types

In java we have things called variables

Variables are used to store data

There are many different types of data

  • Types that store integers (numbers you can count with your fingers) - byte, short, int, and long
  • Types that store numbers with decimals - float and double
  • Type that stores a single letter - char
  • Type that stores whether something is true or false - boolean

if Statements

These are used if you need to make decisions in your program


Ex:

then do this


There are also if else statements


Ex:

if (something is true)

else if (something else is true)

Objects :

Objects are things that have a state and a behavior.

For example we can have a dog object.

Its states could be things such as hair color, breed, and name.

Its behaviors would be things such as barking, eating, or wagging its tail.

Operators

Arithmetic

  • + (Addition) : Adds values
  • - (Subtraction) : Subtracts values
  • * (Multiplication) : Multiplies values
  • / (Division) : Divides values
  • ++ (Increment) : Adds 1 to the value
  • -- (Decrement) : Subtracts 1 to the value

Relational

  • == : Checks if two values are equal
  • != : Checks if two values are not equal
  • > : Checks if the value on the left is greater than the value on the right
  • < : Checks if the value on the left is less than the value on the right
  • >= : Checks if the value on the left is greater than or equal to the value on the right
  • <= : Checks if the value on the left is less than or equal to the value on the right

Loops

A loop allows us to do things multiple times

  • while loop - Repeats something as long as the condition is true
  • for loop - Repeates something for a set amount of times
  • do while loop - Like a while loop except it check if the condition is true at the end instead of the beginning
